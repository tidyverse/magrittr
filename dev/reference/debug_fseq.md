# Debugging function for functional sequences.

This is a utility function for marking functions in a functional
sequence for debbuging.

## Usage

``` r
debug_fseq(fseq, ...)

undebug_fseq(fseq)
```

## Arguments

- fseq:

  a functional sequence.

- ...:

  indices of functions to debug.

## Value

`invisible(NULL)`.
